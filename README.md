# sqldump.py
 
### Usage
Use it to automatise SQL backup with timestamp name. 
To have a automatic backup, I use a cronjob to execute this script every hour.

In the future, I will add a automatic deployment on the cloud

### Installation
```
pip install -r requirements.txt
python3 sqldump.py
```

### Requirements
- A SQL database
- Linux environment (Debian and Ubuntu works fine)
- Python 3.11 (stable version)
